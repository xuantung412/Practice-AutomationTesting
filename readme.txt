How To Run Maven:

1.You need to install Maven on you computer(If you a window user, you need to modify file path your your system...).
2.Go to Project folder and open cmd.
3.Run "mvn -version" to make sure maven is installed in your computer.
4.Then run "mvn-clean" to clean my old maven build.
5.After that run "mvn-install" to rebuild maven project.
6.You can try "mvn exec:java" to run.
